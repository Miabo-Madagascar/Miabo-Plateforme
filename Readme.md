# 🎓 MIABO – Plateforme Éducative Intelligente à Madagascar 🇲🇬

Bienvenue dans le dépôt officiel de **MIABO**, une plateforme numérique moderne qui connecte élèves, tuteurs et parents autour d’un système de tutorat personnalisé basé sur l’intelligence artificielle 🤝📚.

---

## 🌟 Fonctionnalités Clés

- 🔎 **Matching intelligent** élève–tuteur selon les styles d’apprentissage
- 📊 **Dashboard en temps réel** avec graphiques de progression
- 🧠 **Tests IA** de style d’apprentissage & personnalité
- 💬 **Messagerie intégrée** parent ↔ tuteur
- 🧾 **Fiches & syllabus** en ligne pour le suivi pédagogique
- 🎥 **Visioconférence & tableau blanc interactif** (en option)
- 📂 **Base de données éducative** pour l’IA et les ressources

---

## 🧰 Stack Technique

| Frontend          | Backend              | IA        | Database                  |
| ----------------- | -------------------- | --------- | ------------------------- |
| ⚛️ React + Vite | 🚀 FastAPI / Node.js | 🧠 Python | 🗃️ PostgreSQL / MongoDB |

---

## ⚙️ Installation du Projet

> Assurez-vous d’avoir **Node.js ≥ 18** installé.

### 📁 Structure

Miabo-Plateforme/

├── Front/         → Application React (Vite)

├── Back/          → API & logique serveur (FastAPI ou Node.js)

└── README.md


### ▶️ Démarrer le Frontend

```bash
# 1. Clone le dépôt
git clone https://github.com/Miabo-Madagascar/Miabo-Plateforme.git
cd Miabo-Plateforme
# 2. Aller dans le dossier Front
cd Front

# 3. Installer les dépendances
npm install

# 4. Lancer le serveur de développement
npm run dev
